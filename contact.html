Common Questions

Are all tools really free?

Yes — every tool on AxiomApe is 100% free to use with no sign-up required, ever.

Do you store my data?

No. Most tools run entirely in your browser. We do not log or store input data from any tool.

Can I request a new tool?

Absolutely. Use the contact form above, select "Tool Request," and describe what you need.

I found a bug — what should I do?

Please submit a bug report via the form. Include the tool name and a description of what went wrong.
